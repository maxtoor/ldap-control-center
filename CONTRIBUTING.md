# Contributing

Grazie per il contributo a LDAP Control Center.

## Linee guida

- Apri una issue o una discussione prima di cambiamenti strutturali.
- Mantieni le modifiche piccole e focalizzate.
- Aggiorna la documentazione (`README.md`, `CHANGELOG.md`) quando necessario.
